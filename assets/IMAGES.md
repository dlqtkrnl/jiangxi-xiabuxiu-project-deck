# 图片替换说明

当前 `assets` 目录中的 3 个 SVG 文件是为了让网站先具备可展示的视觉结构：

- `hero-material-board.svg`
- `series-look-board.svg`
- `fieldwork-collab-board.svg`

它们不是项目真实照片，而是用于当前面谈版网站的“示意视觉板”。

## 后续最推荐替换的真实图片

### 1. `hero-material-board.svg`

建议替换为：

- 夏布绣细节特写
- 样布纹理照片
- 针法或留白关系近景

### 2. `series-look-board.svg`

建议替换为：

- 你的过往系列作品图
- 概念草图
- 面料实验结果
- 概念板拼图

### 3. `fieldwork-collab-board.svg`

建议替换为：

- 与专家面谈现场照片
- 工作室环境照片
- 田野调研照片
- 示范/工作坊照片

## 替换方法

1. 把真实图片放进 `assets` 目录
2. 保持同样文件名，或修改 `index.html` 中对应的 `img src`
3. 建议横版图片宽度不小于 `1600px`
4. 如果要在手机上也好看，尽量使用裁切明确、主体清晰的图
